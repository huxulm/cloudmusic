## 已支持API
 
|接口名称|URL |
| :--- | :--- |
|手机号登录|LOGIN_CELLPHONE|
|邮箱登录|LOGIN_EMAIL|
|刷新登录|LOGIN_REFRESH|
|登录状态|LOGIN_STATUS|
|发送验证码|CAPTCHA_SENT|
|验证验证码|CAPTCHA_VERIFY|
|退出登录|LOGOUT|
|用户详情|USER_DETAIL|
|用户收藏计数|USER_SUBCOUNT|
|用户等级信息|USER_LEVEL|
|用户歌单|USER_PLAYLIST|
|歌单详情|PLAYLIST_DETAIL|
|歌曲链接|SONG_URL|
|歌曲详情|SONG_DETAIL|
|歌词|SONG_LYRIC|
|搜索|SEARCH|
|精品歌单标签|PLAYLIST_HIGHQUALITY_TAGS|
|喜欢音乐列表|LIKE_LIST|
|喜欢音乐|LIKE|
|每日推荐歌单|RECOMMEND_RESOURCE|
|每日推荐歌曲|RECOMMEND_SONGS|
